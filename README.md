# VueBlogs
Vue js and firebase interaction

---
# Requirement

---
1. node -6.10.x

2. npm -3.10.x
---
Steps to Setup

```
git clone https://github.com/zakiziko/VueBlogs.git

change the link in AddBlog & Blogs to the link of your own firebase

cd VueApp
npm install
npm run dev
```

Vue Serve will run at port http://localhost:8081
